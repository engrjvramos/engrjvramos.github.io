Something about me

Hello, my name is Jose Ramos.

I'm a self-taught web developer & programmer who loves building high quality & responsive websites and web apps.

I graduated in 2017 from Technological Institute of the Philippines with a B.S. in Civil Engineering. I've had the privilege of working with my father's contracting business and with a huge corporation.

My interest in coding started in late 2021 due to my curiosity in building websites and mobile apps. I took a leap and studied modern web development courses on Codecademy and Udemy. I quickly absorbed new concepts, while bringing a positive attitude to every challenge.

Netflix Clone

A lightweight version of Netflix built with NextJS, TypeScript, Firebase and TailwindCSS that allows users to sign up and login, view movie details and trailers, select and purchase plans, and cancel or update membership.

  • Implemented user authentication and account creation functionality.
  • Incorporated Stripe API for handling the payment system and cancelling or updating the user's membership plan.
  • Users can add and remove movies to their "My List".

Presello Clone

A real-estate web app built with React, Firebase and TailwindCSS that allows users to sign up and login to create and save their property listings permanently.

  • Implemented user authentication and account creation functionality.
  • Users can create, edit, save and delete their own property listings.
  • Implemented the Geolocation API and LeafletJS to display the location map of the property listings.
